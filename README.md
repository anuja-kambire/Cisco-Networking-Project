# Cisco Packet Tracer Networking Project

This project demonstrates the design and configuration of a multi-router and multi-switch enterprise network using Cisco Packet Tracer.

## 🔧 Features

* VLAN Configuration & Inter-VLAN Routing
* OSPF & Static Routing
* DHCP Configuration
* Standard & Extended ACLs (Access Control Lists)
* EtherChannel (Link Aggregation)
* Wireless Access Point Setup

## 📊 Description

This project showcases real-world networking concepts such as network segmentation, dynamic routing, IP address management, and security using ACLs. It also includes verification through successful end-to-end connectivity testing.
